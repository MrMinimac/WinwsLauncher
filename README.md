# WinwsLauncher

WinwsLauncher был написан для удобного запуска и настройки инструмента обхода DPI bol-van/zapret (winws.exe).

Репозиторий, с которого взята папка bin (winws.exe, WinDivert.dll) - https://github.com/bol-van/zapret

WinwsLauncher — это небольшое приложение для Windows, которое размещается в системном трее и позволяет вам:

1. Запускать и останавливать обход всего в 2-3 клика.
2. Быстро менять набор параметров запуска через контекстное меню.
3. Настроить автозапуск вместе с Windows.

При первом запуске приложение создает папку в %appdata%/Winws Launcher и скачивает архив winws.zip с этого репозитория, распаковывет в папку appdata%/Winws Launcher

Папка bin/: Здесь находятся исполняемые файлы: Содержит winws.exe и нужные библиотеки (.dll, .sys).
Папка lists/: Здесь находятся списки для настройки обхода:
  lists/list-general.txt: Домены, которые должны обрабатываться.
  lists/list-exclude.txt: Домены, к которым не будут применяться методы обхода.
  lists/ipset-all.txt, lists/ipset-exclude.txt: Аналогичные списки, но для IP-адресов.
Папка args/: Здесь находятся списки с аргументами, которые нужны для запуска winws.exe
